# Template Selenium UI Automate Framework

## Prerequisite

- Visual Studio Code.
- NodeJS.
- Run on cmd "npm install" to install node modules.

## How to run scripts

- npm run qc

## Reports

- Will generate 3 types of reports: xml, json, and html.
- Reports location path: Reports/
- Date format: YYMMDD_HHMMSS
